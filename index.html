App is a helper for grading in 6-grades grading system, it helps to calculate average grade and calculate grade from assignment points.

It also has preferences view, where user can set values for grades.

First view - Average

View has a actionbar with three tabs - first tab is selected, vertical layout whitch has
  • label for input field
  • input field for grades
  • antoher label displaying average grade.

Toast

When user types incorrect grades a toast is displayed. (Maybe its better to restrict programmatically to enter only grades A to F.)

Second view - Grading

View has a actionbar with three tabs - second tab is selected, vertical layout whitch has:
  • label for "Max points" input field
  • input field for max points
  • label for "Points" input field
  • input field for points
  • text field displaying calculated grade and percent.

Third view - Preferences

View has a actionbar with three tabs - third tab is selected, vertical layout whitch has:
  • label "Prefernces"
  • text fields for every grade
  • input field for every grade value
  • OK button for saving.
